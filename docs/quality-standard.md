## 一、验收标准总览

| 指标 | 目标值 | 验收方式 |
|------|--------|----------|
| **SLA** | 99.9% | 每月统计可用时间 |
| **崩溃恢复** | <30s | 故障注入测试 |
| **消息延迟** | <100ms | P95延迟监控 |
| **消息投递率** | 100% | 死信队列监控 |
| **消息重复率** | 0% | messageId去重验证 |

## 二、契约验收标准

### 2.1 Meeting 接口验收

| 字段 | 验收要求 | 测试方法 |
|------|----------|----------|
| `id` | UUID格式，全局唯一 | 正则验证 + 唯一性检查 |
| `version` | 契约版本号，用于兼容检查 | 版本匹配测试 |
| `status` | 七步状态流转正确 | 状态机测试 |
| `createdAt/updatedAt` | 时间戳毫秒精度 | 时间格式验证 |
| `participants` | 政协/人大的Agent ID正确 | 身份映射测试 |
| `decisions` | 投票记录完整 | 决策完整性测试 |

### 2.2 Message 接口验收

| 字段 | 验收要求 | 测试方法 |
|------|----------|----------|
| `id` | messageId唯一，用于去重 | 唯一性检查 |
| `meetingId` | 关联会议存在 | FK约束测试 |
| `type` | 7种消息类型枚举正确 | 枚举测试 |
| `timestamp` | 时间戳毫秒精度 | 时间格式验证 |

### 2.3 StateSnapshot 接口验收

| 字段 | 验收要求 | 测试方法 |
|------|----------|----------|
| `checksum` | 数据完整性校验 | checksum验证测试 |
| `timestamp` | 快照时间戳 | 时间格式验证 |

### 2.4 HealthCheck 接口验收

| 字段 | 验收要求 | 测试方法 |
|------|----------|----------|
| `status` | healthy/degraded/unhealthy枚举 | 枚举测试 |
| `metrics.averageLatency` | <100ms | 性能测试 |
| `metrics.errorRate` | <0.1% | 错误率监控 |

## 三、事件验收标准

### 3.1 事件通道验收

| 通道 | 验收要求 | 测试方法 |
|------|----------|----------|
| `pandaclaw:meeting:events` | 会议事件正确路由 | 事件路由测试 |
| `pandaclaw:agent:events` | Agent事件正确路由 | 事件路由测试 |
| `pandaclaw:system:events` | 系统事件正确路由 | 事件路由测试 |

### 3.2 事件类型验收

| 事件类型 | 验收要求 | 测试方法 |
|----------|----------|----------|
| `meeting:created` | payload包含topic/type/context | payload验证 |
| `step:changed` | payload包含fromStep/toStep/reason | 状态流转测试 |
| `opinion:submitted` | payload包含agentId/opinion/step | 意见提交测试 |
| `stance:recorded` | payload包含stance/reason | 立场标记测试 |
| `inquiry:created` | payload包含questionId/question | 质询测试 |
| `vote:cast` | payload包含vote/reason | 投票测试 |
| `decision:made` | payload包含voteSummary/passed | 决策测试 |

### 3.3 死信队列验收

| 要求 | 验收标准 | 测试方法 |
|------|----------|----------|
| 重试次数 | retryCount < 3 | 重试策略测试 |
| 失败记录 | failureReason明确 | 错误追踪测试 |
| 清理策略 | TTL过期清理 | TTL测试 |

### 3.4 熔断器验收

| 状态 | 验收要求 | 测试方法 |
|------|----------|----------|
| `closed` | 正常运行 | 基线测试 |
| `open` | 错误率>50%触发熔断 | 熔断触发测试 |
| `half-open` | 30s后尝试恢复 | 熔断恢复测试 |
